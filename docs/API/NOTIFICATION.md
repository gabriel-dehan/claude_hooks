## Hook Exit Codes

| Exit Code | Behavior |
|-----------|----------|
| `exit 0` | Operation continues<br/>Logged to debug only (`--debug`) |
| `exit 1` | Non-blocking error<br/>Logged to debug only (`--debug`) |
| `exit 2` | N/A<br/>Logged to debug only (`--debug`) |
